# ab-cv

A modern, automated CV/Resume generation system using LaTeX. This repository provides a clean, professional template for creating, maintaining, and hosting your CV in LaTeX format.

## Features

- **LaTeX-based CV**: Create a professional CV using LaTeX for perfect typography and formatting
- **Automated Compilation**: Build PDF output automatically from your LaTeX source
- **Version Control**: Track all changes to your CV with Git history
- **Easy Customization**: Modular structure makes it simple to personalize your template
- **Hosted Output**: Generated PDF is hosted and easily shareable
- **Responsive Updates**: Update your CV once, deploy everywhere

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **LaTeX Distribution**:
  - Linux: `sudo apt-get install texlive-full` (Ubuntu/Debian)
  - macOS: Install [MacTeX](https://tug.org/mactex/)
  - Windows: Install [MiKTeX](https://miktex.org/) or [TeX Live](https://tug.org/texlive/)
- **Git**: For version control
- **Make** (optional): For convenient build commands
